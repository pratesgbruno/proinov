# PROINOV

Estrutura inicial de um site institucional estático para a PROINOV.

## Arquivos

- `index.html`: página principal
- `assets/css/styles.css`: estilos complementares
- `assets/js/main.js`: interações básicas

## Como abrir

Você pode abrir o `index.html` diretamente no navegador, mas para evitar limitações locais com alguns recursos é melhor rodar um servidor simples:

```bash
cd /Users/brunoprates/Documents/Codigo/proinov
python3 -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```
